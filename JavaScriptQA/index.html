1. Dado un string cree una función que invierta el orden de sus caracteres.

Ejemplo: "Hello World = "dlroW olleH" No puedes usar desestructuración ni recursión

2. Dado un String, Cree una función que invierta el orden de sus caracteres. Deve utilizar un desestructuración en el proceso

3. Dado un String, Invierta el orden de sus caracteres recursivamente

4. Dado un String, Invierta el orden de sus caracteres sin utilizar una nueva string(Realice las modificaciones en el propio string)

5. Construya una función isPalindrome, la cual dado un string nos indique si es un palindrome o no

Ejemplo de palindromes: "Yo dono rosas , oro no doy"

6. Ejercicio Opcional: Por último, intente invertir las palabras de una frase, pero por cada palabra debe mantenerse en el lugar

Ejemplo: "Hello world" = "olleH dlrow"
